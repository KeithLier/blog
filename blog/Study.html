# 小程序

​wx-promise-pro (opens new window)扩展小程序api支持promise，安装

npm i wx-promise-pro -S

在小程序入口（app.js）调用一次promisifyAll，只需要调用一次。 示例:

import { promisifyAll, promisify } from 'wx-promise-pro'

// promisify all wx‘s api
promisifyAll()

// 使用
wxp.login().then(res => {
  // do something
})
